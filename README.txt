ExtraGamemodes by Andrew Pratt
A tool for generating extra gamemodes for Hitman 3.

=VERY MUCH A WORK IN PROGRESS!=

github: https://github.com/andrewpratt64/ExtraGamemodes

Currently only has one gamemode, the "Kill Everyone Challenge".
More gamemodes will be added in the future, along with options for the current gamemode, and more elegant file generation.
